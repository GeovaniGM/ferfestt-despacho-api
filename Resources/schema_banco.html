Tabelas

Usuários do backoffice administrativo. Iniciado com um registro fixo. Senha armazenada como hash bcrypt.

despachos operacional

Registros de despacho confirmados. Imutáveis pelo operador. Soft delete pelo admin via campo ativo. A unique constraint em chave_nfe é parcial — aplica-se apenas a registros ativos, permitindo que uma NFe inativada seja despachada novamente.

ColunaTipoAtributosDescrição
idUUIDPK NOT NULLIdentificador único
bancada_idUUIDFK NOT NULLReferência a bancadas.id
chave_nfeCHAR(44)UK parcial* NOT NULLChave de acesso completa da NFe
numero_nfeVARCHAR(9)NOT NULLNúmero extraído da chave (posições 26–34)
lido_emTIMESTAMPTZNOT NULLTimestamp do dispositivo no momento da leitura
registrado_emTIMESTAMPTZNOT NULL default now()Timestamp do servidor no momento da persistência
ativoBOOLEANNOT NULL default true IDXfalse = inativado pelo admin
* Unique constraint parcial: CREATE UNIQUE INDEX uq_despachos_chave_ativa ON despachos (chave_nfe) WHERE ativo = true;
Garante que não existam dois despachos ativos com a mesma chave, mas permite que uma chave inativada seja despachada novamente.

despacho_eventos auditoria

Log unificado de todos os eventos do ciclo de vida de um despacho: tentativas de sincronização (aceitas ou rejeitadas) e inativações pelo admin. despacho_id é nulo para eventos do tipo rejeitado, pois nenhum despacho foi criado.

SQL de criação

CREATE EXTENSION IF NOT EXISTS "pgcrypto";

CREATE TABLE usuarios (
  id          UUID         PRIMARY KEY DEFAULT gen_random_uuid(),
  email       VARCHAR(255) NOT NULL UNIQUE,
  senha_hash  VARCHAR(255) NOT NULL,
  ativo       BOOLEAN      NOT NULL DEFAULT true,
  criado_em   TIMESTAMPTZ  NOT NULL DEFAULT now()
);

CREATE TABLE bancadas (
  id          UUID         PRIMARY KEY DEFAULT gen_random_uuid(),
  codigo      VARCHAR(50)  NOT NULL UNIQUE,
  descricao   VARCHAR(255) NOT NULL,
  ativo       BOOLEAN      NOT NULL DEFAULT true,
  criado_em   TIMESTAMPTZ  NOT NULL DEFAULT now()
);

CREATE TABLE despachos (
  id             UUID        PRIMARY KEY DEFAULT gen_random_uuid(),
  bancada_id     UUID        NOT NULL REFERENCES bancadas(id),
  chave_nfe      CHAR(44)    NOT NULL,
  numero_nfe     VARCHAR(9)  NOT NULL,
  lido_em        TIMESTAMPTZ NOT NULL,
  registrado_em  TIMESTAMPTZ NOT NULL DEFAULT now(),
  ativo          BOOLEAN     NOT NULL DEFAULT true
);

CREATE UNIQUE INDEX uq_despachos_chave_ativa
  ON despachos (chave_nfe)
  WHERE ativo = true;

CREATE INDEX idx_despachos_bancada_id  ON despachos (bancada_id);
CREATE INDEX idx_despachos_registrado  ON despachos (registrado_em);
CREATE INDEX idx_despachos_ativo       ON despachos (ativo);

CREATE TABLE despacho_eventos (
  id             UUID        PRIMARY KEY DEFAULT gen_random_uuid(),
  despacho_id    UUID        REFERENCES despachos(id),
  bancada_id     UUID        NOT NULL REFERENCES bancadas(id),
  chave_nfe      CHAR(44)    NOT NULL,
  tipo           VARCHAR(20) NOT NULL CHECK (tipo IN ('sincronizado','rejeitado','inativado')),
  tentativas     INTEGER,
  enviado_em     TIMESTAMPTZ,
  respondido_em  TIMESTAMPTZ,
  erro           VARCHAR(50),
  motivo         TEXT,
  criado_em      TIMESTAMPTZ NOT NULL DEFAULT now()
);

CREATE INDEX idx_eventos_despacho_id ON despacho_eventos (despacho_id);
CREATE INDEX idx_eventos_chave_nfe   ON despacho_eventos (chave_nfe);
CREATE INDEX idx_eventos_tipo        ON despacho_eventos (tipo);
CREATE INDEX idx_eventos_bancada_id  ON despacho_eventos (bancada_id);
